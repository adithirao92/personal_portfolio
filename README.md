Personal Portfolio Website
This is my personal portfolio website built with React to showcase my projects, skills, and experiences. The website includes a clean and modern design with sections for:

About Me – A brief introduction about myself and my background.

Projects – A list of projects I’ve worked on with detailed descriptions.

Skills – A display of technical and soft skills.

Contact – A section for reaching out to me via social media or email.

Tech Stack
Frontend – React, HTML, CSS

Styling – Custom CSS for a responsive and modern design

Features
Clean and responsive design

Professional header with dynamic content

Organized and easy-to-navigate sections

Mobile-friendly layout
